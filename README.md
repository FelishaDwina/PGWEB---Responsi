# WebGIS Pemetaan Pelabuhan Sulawesi Tenggara

## Deskripsi Produk
WebGIS Pemetaan Pelabuhan Sulawesi Tenggara merupakan aplikasi berbasis web yang menyajikan informasi lokasi pelabuhan secara interaktif dalam bentuk peta digital. Aplikasi ini menampilkan data spasial pelabuhan beserta informasi pendukung seperti nama pelabuhan, jenis pelabuhan, lokasi administratif, status operasional, deskripsi, serta gambar pelabuhan. WebGIS ini dikembangkan sebagai bagian dari tugas Pemrograman Geospasial Web.

## Teknologi Pembangun
Aplikasi WebGIS ini dibangun menggunakan beberapa teknologi, yaitu:
- PHP
- MySQL
- Leaflet JS
- Bootstrap
- HTML
- CSS
- XAMPP sebagai web server lokal

## Sumber Data
Data yang digunakan dalam WebGIS ini meliputi:
- Data koordinat pelabuhan (latitude dan longitude)
- Data atribut pelabuhan (nama, jenis, lokasi, status, dan deskripsi)
- Data gambar pelabuhan  
Data diperoleh dari pengolahan dan input manual sebagai data pendukung dalam kegiatan praktikum.
